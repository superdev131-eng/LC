Simply outputs an XML Header tag.

Statamic's template parser will encode `<?` and `?>` because they are PHP tags. Silly XML. This global will get you around that.

```
{{ xml_header }}
```

``` .language-output
<?xml version="1.0" encoding="utf-8" ?>
```
