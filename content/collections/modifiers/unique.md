title: Unique
---
Returns all of the unique items in the array:

```yaml
checklist:
  - zebra
  - hippo
  - hyena
  - giraffe
  - zebra
  - hippo
  - hippo
  - hippo
  - hippo

```

```
{{ checklist | unique | list }}
```

```html
zebra, hippo, hyena, giraffe
```
