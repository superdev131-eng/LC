title: UL

Turn an array into an HTML unordered list element.

```yaml
food:
  - sushi
  - broccoli
  - kale
```

```
{{ food | ul }}
```

```html
<ul>
  <li>sushi</li>
  <li>broccoli</li>
  <li>kale</li>
</ul>
```
