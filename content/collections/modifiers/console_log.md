title: 'Console Log'

Debug a variable by dumping its contents to your browser's JavaScript's console via `console.log`.

```yaml
fruit:
  - apples
  - bananas
  - bacon
```

```
{{ fruit | console_log }}
```

```js
["apples", "banana", "jerky"]
```
