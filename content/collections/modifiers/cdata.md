title: CDATA
---
Wraps a string in [CDATA][cdata] tags, useful for formatting characters properly in XML.

```yaml
title: My Very Own Podcast
```

```
{{ title | cdata }}
```

```html
<![CDATA[My Very Own Podcast]]>
```

[cdata]: https://en.wikipedia.org/wiki/CDATA
