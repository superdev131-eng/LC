title: 'Has Lower Case'
---
Returns `true` if the string contains a lowercase character, `false` otherwise.

```yaml
loud_noises: "I DON'T KNOW WHAT WE'RE YELLING ABOUT!"
```

```
{{ if loud_noises | has_lower_case }}
```

```html
false
```
