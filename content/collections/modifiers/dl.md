Turn a key/value array, otherwise known as a YAML mapping, into an HTML definition list.

```yaml
food:
  Delicious:
    - bacon
    - sushi
  Green:
    - broccoli
    - kale
```

```
{{ food | dl }}
```

```html
<dl>
  <dt>Delicious</dt>
  <dd>bacon</dd>
  <dd>sushi</dd>

  <dt>Green</dt>
  <dd>broccoli</dd>
  <dd>kale</dd>
</dl>
```
