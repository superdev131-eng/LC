title: 'Is Numeric'

Returns `true` if variable is a number or numeric string.

```yaml
sequence: 4815162342
another_sequence: just type 4 8 15 16 23 42
```

```
{{ if sequence | is_numeric }}
{{ if another_sequence | is_numeric }}
```

```html
true
false
```
