Ensures that the string never begins with a specified string.

```.language-yaml
twitter: @statamic
```

```
<a href="http://twitter.com/{{ twitter | remove_left:@ }}">Twitter</a>
```

```.language-output
<a href="http://twitter.com/statamic">Twitter</a>
```
