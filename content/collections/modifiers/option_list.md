title: 'Option List'

Turn an array into a pipe-delimited string. Useful when passing an array of things into a parameter.

```yaml
collections:
  - blog
  - news
  - wigs
```

```
{{ collection from="{collections|option_list}" }}
```

Can also be used by its alias, [`piped`](/modifiers/piped).
