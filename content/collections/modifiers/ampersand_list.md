title: 'Ampersand List'
---
Turn a simple array into a comma delimited string with a friendly little ampersand between the last two items.

```yaml
fruits:
  - apples
  - bananas
  - jerky
```

```
{{ fruits | ampersand_list }}
```

```html
apples, bananas & jerky
```
