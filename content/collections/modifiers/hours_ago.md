title: 'Hours Ago'
---
Returns the number of hours since a given date variable. Statamic will attempt to parse any string as a date, but try to keep it in the least ambiguous date format possible.

```yaml
date: October 1 2015
```

{{ noparse }}
```
{{ date | hours_ago }}
```
{{ /noparse }}

```html
{{ test_date | hours_ago }}
```
