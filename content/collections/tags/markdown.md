This is a tag-pair version of the [markdown modifier](/modifiers/markdown).

## Example {#example}

```
{{ markdown }}
## Ode To Bacon

Bacon. Mmm, bacon.
{{ /markdown }}
```

``` .language-output
<h3>Ode To Bacon</h3>
<p>Bacon. Mmm, bacon.</p>
```
